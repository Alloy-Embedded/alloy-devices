- **espressif/esp32s3**: Dual-core control plane (PRO_CPU + APP_CPU) is emitted via `bring_up_app_cpu()` toggling `SYSTEM.CORE_1_CONTROL_*` registers.  Per-interrupt affinity routing and inter-core IPC primitives are delegated to esp-idf.
- **microchip/avr-da**: 8-bit Harvard memory layout (flash @ 0x0, SRAM @ 0x4000); host-ld validation is skipped and the vendor toolchain (avr-gcc) is the canonical link gate.  Vector table follows the avr-libc `__vector_<n>` convention.
- **raspberrypi/rp2040**: Dual-core Cortex-M0+ modelled as single-core perspective — the emitted startup brings up core 0 only.  Core 1 bring-up and inter-core FIFO/spinlock primitives are deferred to a follow-on proposal.
